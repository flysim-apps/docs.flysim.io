# Hangar & Fleet Management
{: .no_toc }

## Table of contents
{: .no_toc .text-delta }

1. TOC
{:toc}

---

The **Hangar** is where you manage all the aircraft in your virtual fleet. Every aircraft you fly in FlyAround must first be added to your Hangar.

---

## Opening the Hangar

Click **Hangar** in the main navigation menu.

---

## Browsing Your Fleet

Aircraft are displayed in a masonry grid — cards showing each aircraft's livery image, name, and key stats. The default sort is by total flight count.

### Search and Filter

| Control | Description |
|---------|-------------|
| **Search box** | Filter by aircraft type code, airline name, or tail number |
| **Loaded aircraft toggle** | Show only the aircraft currently loaded in your simulator |
| **Refresh** | Reload fleet data from the server |

---

## Adding an Aircraft to Your Fleet

{: .highlight }
You must add at least one aircraft before you can dispatch your first flight.

1. Open the **Hangar**
2. Click the **"Add Aircraft"** button
3. Enter the aircraft details:
   - **Airframe** — Aircraft type (e.g., B737, A320)
   - **Airline** — Livery/airline you are flying
   - **Tail Number** — Registration (e.g., N12345, G-ABCD)
4. Save — the aircraft appears in your Hangar and is available in the Dispatcher

---

## Aircraft Detail View

Click any aircraft card to open its full detail page.

### Information Panel

| Field | Description |
|-------|-------------|
| Aircraft Image | Full livery artwork |
| Aircraft Model | Type and variant name |
| Tail Number | Registration / N-number |
| Airline | Airline name and IATA code |
| Total Flights | Number of completed flights with this aircraft |
| Flight Hours | Total Hobbs time |

### Routes Map

An interactive map showing all routes you have flown with this aircraft — route lines connect origin and destination airports.

### Recent Flights Table

The 5 most recent completed flights with this aircraft:
- **Callsign** (click to open full flight details)
- **Route** (origin → destination)
- **Date**

---

## Removing an Aircraft

In the Aircraft Detail View, click **Delete Aircraft**. Confirm in the dialog that appears.

{: .note }
Deleting an aircraft does **not** delete your flight history — Logbook entries are preserved.

---

## Tips

- Use consistent tail number naming to keep your fleet organized.
- The **Loaded Aircraft** filter is useful for large fleets — quickly find what's in your sim right now.
- If an aircraft appears with no image, verify the airline and airframe codes are correctly configured.
